# Nümtema Pack Factory - Project Brief

## Intent
Build a unified vertical mission engine using PocketFlow that extracts requirements, conducts a clarification interview, structures the mission specification, executes generating deliverables, and runs verification gates. The primary vertical to implement is the Procedure Pack.

## Objectives
1. Implement a clean `pocketflow.py` local module.
2. Develop `procedure_flow.py` matching the common core workflow.
3. Build a FastAPI backend and a gorgeous, glassmorphic UI dashboard to interact with the engine.
4. Verify the outputs against structural gates.
